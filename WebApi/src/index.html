Here, the ideia is show how we could bring the ideia of an Asp.Net WebApi lookalike, inside the Node.js

The firt step for this, was import the Express inside our project, so this way we're able to set a simple server running in our localhost.

The next steps, you may guessing, could be, how to solve DI, how to solve the Controllers.
Well, we'll gonna try to show how this could be setting up, in a simple way.

The first thing to do, is define our structure here, once Node.js dosen't have this concepts, and as Sap said, you could leave this way, or create this foundations, if it'll let you more confortable as an Asp.Net Core dev.
